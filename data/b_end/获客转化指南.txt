# 商家获客与转化指南

## 一、获客渠道概述

### 1.1 平台内获客

**搜索流量**
- 优化商品标题和描述
- 提高店铺评分
- 积累好评数量

**推荐流量**
- 参与平台活动
- 保持店铺活跃度
- 提升转化率

**付费推广**
- 搜索广告
- 信息流广告
- 活动专题位

### 1.2 平台外获客

**社交媒体**
- 微信朋友圈
- 小红书种草
- 抖音短视频

**线下引流**
- 门店二维码
- 活动物料
- 老客户转介绍

## 二、客户触达技巧

### 2.1 最佳触达时间

**业主客户**
| 时段 | 推荐度 | 原因 |
|------|--------|------|
| 10:00-11:30 | ⭐⭐⭐⭐⭐ | 上午工作间隙，相对空闲 |
| 14:00-15:30 | ⭐⭐⭐⭐⭐ | 午休后精神好，决策能力强 |
| 19:30-21:00 | ⭐⭐⭐⭐ | 晚间休息，有时间详细了解 |
| 12:00-14:00 | ⭐ | 午休时间，容易打扰 |
| 22:00后 | ❌ | 太晚会引起反感 |

### 2.2 话术模板

**首次接触**
```
您好！看到您在关注[品类]，我是洞居平台的专业顾问，可以为您提供一些选购建议。

我们平台有丰富的优质商家资源，而且现在还有补贴活动，可以帮您省不少钱。

方便的话，可以告诉我您的具体需求，我来帮您匹配最合适的方案。
```

**跟进话术**
```
您好！之前和您聊过[品类]的事情，想了解一下您考虑得怎么样了？

最近平台有一些新的优惠活动，我觉得挺适合您的情况。

如果您有任何疑问，随时可以问我，我来帮您解答。
```

**促单话术**
```
您好！您之前看中的[品类]，现在正好有限时优惠活动。

这个活动力度很大，而且名额有限，错过可能要等很久。

如果您决定好了，我可以帮您锁定优惠名额。
```

### 2.3 沟通技巧

1. **倾听为主**：先了解客户需求，再推荐产品
2. **专业解答**：用专业知识建立信任
3. **不急于成交**：给客户思考空间
4. **持续跟进**：保持适度联系，不要过于频繁

## 三、转化率优化

### 3.1 转化漏斗分析

```
曝光 → 点击 → 咨询 → 到店 → 成交
100%   10%    3%     1%    0.5%
```

### 3.2 各环节优化

**提高点击率**
- 优化主图，突出卖点
- 标题包含热搜关键词
- 价格有竞争力

**提高咨询率**
- 详情页信息完整
- 突出优惠信息
- 设置咨询入口

**提高到店率**
- 快速响应咨询
- 提供专业建议
- 预约到店优惠

**提高成交率**
- 到店体验好
- 报价透明
- 售后有保障

## 四、ROI分析与优化

### 4.1 ROI计算公式

```
ROI = (收入 - 投入) / 投入 × 100%
```

### 4.2 ROI评估标准

| ROI | 评级 | 建议 |
|-----|------|------|
| ≥200% | 优秀 | 继续保持，可扩大投入 |
| 100-200% | 良好 | 适度增加投入 |
| 50-100% | 一般 | 优化投放策略 |
| 0-50% | 较低 | 分析原因，调整策略 |
| <0% | 亏损 | 暂停投放，重新评估 |

### 4.3 优化建议

**ROI偏低时**
1. 检查投放渠道精准度
2. 优化目标人群定向
3. 分析转化漏斗，找出流失环节
4. 测试不同的投放时段和素材

**ROI良好时**
1. 考虑扩大投放规模
2. 尝试拓展新渠道
3. 优化高转化素材
4. 提高客单价

## 六、常见问题

### Q1: 新店如何快速获客？
A: 建议先参与平台新商家扶持活动，同时做好基础优化（商品、详情页、评价），再逐步增加付费推广。

### Q2: 推广费用如何控制？
A: 建议设置日预算上限，先小规模测试，找到效果好的渠道和素材后再加大投入。

### Q3: 如何提高客户复购率？
A: 做好售后服务，建立客户微信群，定期推送优惠信息，老客户专属福利。
